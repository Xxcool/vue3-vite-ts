## 说明

使用最新的前端技术栈 Vue3、Vite3、TypeScript、NaiveUI、Pinia 和 UnoCSS

- 克隆代码

```bash
git clone https://github.com/Xxcool/study.git
```

- 安装依赖

```bash
pnpm i
```

- 运行

```bash
pnpm vite
```

- 打包

```bash
pnpm build
```
